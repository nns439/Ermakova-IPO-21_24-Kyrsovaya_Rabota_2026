# Веб-приложение подбора тарифного плана МТС

Учебный проект на Python + Flask + SQLite. Приложение помогает клиенту подобрать тариф по стоимости, минутам, SMS, мобильному интернету, домашнему интернету и дополнительным услугам. Есть роли администратора, менеджера и пользователя.

## Возможности

- авторизация пользователей с ролями: `admin`, `manager`, `user`;
- каталог из 20 тестовых тарифных планов;
- подбор тарифа по взвешенной сумме критериев;
- сравнение нескольких тарифов;
- история подборов и история просмотров;
- CRUD для тарифов, истории подборов, категорий клиентов и дополнительных услуг;
- аналитика: частота выбора тарифов и распределение предпочтений по категориям;
- SQLite-база с SQL-скриптами `schema.sql` и `seed.sql`;
- запуск через Docker и Docker Compose.

## Демо-аккаунты

| Роль | Логин | Пароль |
| --- | --- | --- |

## Локальный запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

После запуска откройте: <http://localhost:5000>

## Запуск через Docker

```bash
docker compose up --build
```

Приложение будет доступно по адресу: <http://localhost:5000>

SQLite-файл хранится в Docker volume `mts_sqlite_data`.

## Структура проекта

```text
app.py                 # серверная логика Flask
schema.sql             # создание таблиц, ключей, индексов
seed.sql               # тестовые данные, включая 20 тарифов
templates/             # HTML-шаблоны
static/css/style.css   # оформление интерфейса
static/js/app.js       # небольшие улучшения интерфейса
Dockerfile             # контейнеризация приложения
docker-compose.yml     # запуск приложения с volume для SQLite
```
